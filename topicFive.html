Local Storage

You can access your local storage by using localStorage syntax. But you may not know what it is? Well, it's a way of storing data locally on your browser rather than via cookies. What's interesting about this way of storing data is the fact that it will stay with you even after you close your brower session. Although we are able to have data store for just a session if we desired, we'd just have to use sessionStorage instead!

Either way, you might be wondering how we can even store information into local stoarge? Well that part is actually pretty easy. The way we store inforamtion into local storage is by using the following format.

setItem

        localStorage.setItem("key", "value");.

You choose a key which is like an variable or a name, and a value to be stored into it. So for example if I did this: 

          localStorage.setItem("school", "BYUI");

getItem

The above line of code will actually store BYUI into the key "school". Now that I have it stored, I need to access/use it. How can that be done? Pretty much the same way, only instead of using setItem, we'll use getItem. Below is how I would display/access my stored value. 

          document.getElementById("example1").innerHTML = localStorage.getItem("school");

We can now see what information was stored in the local storage, as you can tell, that would be BYUI. Now, there is another way to either store data or change what is stored and it's very similar, we can also do this:

Alternative way to set a value/change a value.

        localStorage.school = "UNM";

Here we are setting localStorage.school to hold the value UNM in replace of BYUI, now if we were to try to display using the same previous line, it would instead say UNM rather than BYUI.

removeItem

We can also remove the item all together, to do this we use removeItem instead just like so: 

localStorage.removeItem("school");

Notice here if we tried to display what was in school, nothing would appear. I've now removed that key and it's value all together.

Arrays

We can also use local storage with an array. Unfortnatly you'll also need to convert the array using JSON stringify.

For this example, let's say I have a list of crypto currencies for example.. (yes I love crypto).

      var cryptoList ["Bitcoin", "Litecoin", "XRP", "Holo", "BNB", "LINK", "ZIL"];

Now we need to convert this JS array into JSON format using stringify. This can be done like this: 

      JSON.stringify(cryptoList);

Now we just combine the two like so

        var cryptoList = ["Bitcoin", "Litecoin", "XRP", "Holo", "BNB", "LINK", "ZIL"];

        localStorage.setItem("cryptoList", JSON.stringify(cryptoList));

That's how you would use an array in local storage. Now to just access/display it we would do the following.

 document.getElementById("example1").innerHTML = localStorage.getItem("cryptoList");

sessionStorage

As previously meantioned I just wanted to renforce the fact that all the same logic applies to storing information into a session instead of local storage. The difference is it's gone after the browser is closed. To do this you would instead use "sessionStorage" in replace of localStorage.

Example1

        //create key and value
        localStorage.setItem("school", "BYUI");

        //change value  to UNM
        localStorage.school = "UNM";

        //remove it all together (pointless but for example)
        localStorage.removeItem("school");

        //create the value again, but this time it set to ASU
        localStorage.school = "ASU";

        //display school.
        document.getElementById("example1").innerHTML = "School: " + localStorage.getItem("school");

Example2

//example 2
        //creat JS object
        var cryptoList = ["Bitcoin", "Litecoin", "XRP", "Holo", "BNB", "LINK", "ZIL"];

        //store it in local storage, under the key crypto list, and set the value to the JSON converstion of our JS object crypto list
        localStorage.setItem("cryptoList", JSON.stringify(cryptoList));

        //display
        document.getElementById("example2").innerHTML = localStorage.getItem("cryptoList");
